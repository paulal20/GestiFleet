HTML
Lenguaje de marcado para estructurar páginas web. Utiliza etiquetas para definir y organizar elementos como textos, imágenes, enlaces y vídeos, permitiendo que los navegadores interpreten y muestren la información al usuario.
Markup language to structure web pages. It uses tags to define and organize elements like text, images, links, and videos, allowing browsers to interpret and display information to the user.
CSS
Lenguaje de estilos para dar formato al contenido. Permite controlar cómo se ven los elementos de una página, como los colores, las fuentes, los márgenes y la disposición del contenido, haciendo que la web sea más atractiva y fácil de usar.
Style sheet language to format content. It controls how page elements look, such as colors, fonts, margins, and layout, making the web more attractive and user-friendly.
JSON
Formato de texto ligero y legible por humanos para almacenar e intercambiar datos, basado en la sintaxis de objetos de JavaScript pero independiente de cualquier lenguaje de programación. Se basa en la sintaxis de los objetos de JavaScript, pero es un formato independiente del lenguaje, lo que permite su uso en una amplia variedad de lenguajes de programación. Sus principales usos son la comunicación entre servidores y aplicaciones, la configuración de software y el almacenamiento de datos en bases de datos NoSQL.
Lightweight, human-readable text format for storing and exchanging data, based on JavaScript object syntax but independent of any programming language. It allows use across many programming languages. Main uses include communication between servers and applications, software configuration, and data storage in NoSQL databases.
